# clean_architecture_flutter

A Flutter project With Clean Architecture + Bloc + MVVC

## Structure

1- Domain
    => Entity
    => repositories
    => usecase
2-Data
   => Models
   => repositories
   => datasources
3-Presentation
  => Bloc + State
  => Screens
  => Widgets
